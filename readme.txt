
MACHINE LEARNING PROJECT

Classifying Ephemeral vs Evergreen Content on the Web


->Go to Script folder
->Commands to install libraries required (Use virtual environment):
	$ virtualenv -p /usr/bin/python project
	$ source project/bin/activate
	$ pip install numpy;pip install pandas;pip install scipy;pip install sklearn;pip install nltk
->Installing NLTK libraries required:
	$ python
	$ import nltk
	$ nltk.download()
	-> USE nltk's GUI to download stopwords, punkt, wordnet packages from the list in all packages tab

->running the program:
	$ python main.py
	->this will generate many files whose description is provided in report
